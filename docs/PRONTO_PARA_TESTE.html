🚀 Como Testar (3 Passos Rápidos)

ℹ️ O servidor Node.js já está rodando. Os dados de teste já foram inseridos!
1️⃣ Abra o Navegador

Abra a página de login de teste:

file:///home/aluno/Documentos/TCC-Posto-de-Coleta/login/login-teste.html
2️⃣ Faça Login como Admin ou Médico

Selecione "Admin" ou "Médico" e clique "Entrar"

⚠️ Se for "Paciente", verá aviso de acesso restrito

3️⃣ Visualize os Pacientes

Você será redirecionado para a página de pacientes com:

  • Tabela com 5 pacientes de teste
  • Estatísticas atualizadas
  • Funcionalidade de busca
  • Modal com detalhes completos
