🐛 PROBLEMAS COMUNS E SOLUÇÕES:
═══════════════════════════════════════════════════════════════════════════

PROBLEMA: "Nenhum paciente encontrado"
SOLUÇÃO: Rodar script de teste:
  ./scripts/inserir-dados-teste.sh

PROBLEMA: "Aviso de acesso restrito" (mesmo sendo admin)
SOLUÇÃO: Definir usuário no console:
  definirUsuarioTeste('admin')

PROBLEMA: "Tabela aparece desabilitada"
SOLUÇÃO: Verificar tipo de usuário:
  console.log(auth.obterUsuario().tipo)
  # Deve ser 'admin' ou 'medico', não 'paciente'

PROBLEMA: "Botão não funciona" ou "Modal não abre"
SOLUÇÃO: Verificar console (F12) por erros em vermelho

PROBLEMA: "Página em branco"
SOLUÇÃO: 
  1. Verificar se servidor está rodando: npm run dev
  2. Verificar se MongoDB está rodando
  3. Abrir console (F12) e ver erro
